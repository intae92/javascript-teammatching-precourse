## 📜 구현할 기능 목록
- [ ] 다른 탭으로 이동했다 돌아와도 기존 탭의 상태가 유지되어야 한다.
- [ ] localStorage이용하여 최근 작업 정보 유지 해야 한다.
- 탭 메뉴
  - [ ] 메뉴 클릭 시 다른 탭으로 이동 해야 한다
- 크루 관리 기능 탭
  - [ ] 코스별로 크루를 추가할 수 있다.
    - [ ] 동일한 이름의 크루는 추가 할 수 없다.
    - [ ] 크루 이름은 최대 1~5글자까지 가능하다.
  - [ ] 코스별로 크루를 삭제할 수 있다.
    - [ ] 삭제할 때는 confirm을 사용하여 사용자가 한번 더 확인해야한다.
- 팀 매칭 관리 기능
